Install Flask-Servatus
======================

.. include:: header.rst

Install Flask-Servatus via pip
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Flask Servatus is available on PyPi and can be installed using a python package manage like pip

.. sourcecode:: shell

    $ pip install Flask-Servatus


or via easy install


.. sourcecode:: shell

    $ easy_install Flask-Servatus


Install Flask-Servatus from source
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

.. sourcecode:: shell

    $ git clone https://github.com/mikeywaites/Flask-Servatus.git flask_servatus
    $ python setup.py install
